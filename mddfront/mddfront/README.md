# MDD Front-end (Monde de Dev)

**Author:** João Paulo SEBASTIAO  
**Version:** 0.0.1  
**Technologies:** Angular 21, TypeScript, RxJS, Angular Material  

## Project Description

MDD (Monde de Dev) is an internal social network for developers. This front-end application allows users to:  

- Register and log in securely using JWT authentication.  
- View and edit user profiles.  
- Browse and create themed posts (articles).  
- View and add comments to posts.  
- Subscribe and unsubscribe to topics of interest.  
- Access a personalized news feed based on subscriptions.  

The project uses Angular standalone components for a modern, modular, and testable architecture.
